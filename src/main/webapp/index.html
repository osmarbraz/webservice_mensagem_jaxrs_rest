Recurso Mensagem JAX-RS REST

Serviço getMensagem

URL: rest/mensagem

Comando: curl http://localhost:8080/webservice_mensagem_jaxrs_rest/rest/mensagem

Serviço setMensagem(string)

Enviar dados através de formulário HTML: setmensagem

Comando: curl -v -X POST -H "Content-Type: application/json" -d "{\"mensagem\":\"Nova Mensagem\"}" http://localhost:8080/webservice_mensagem_jaxrs_rest/rest/mensagem
